# Eidos 🧠✨
Your AI memory companion — part fun friend, part productivity buddy.  
Built with Flutter + Supabase + Gemini.

---

## 🚀 What it does
- Chat with an AI that **remembers everything** you say.
- Structured memory: goals, facts, preferences → never forgotten.
- Proactive nudges: reminders, callouts, and accountability.
- Personality layer: warm, witty, and calls out your BS kindly 😉.
- Fully private: all requests go through your backend with per-user limits.

---

## 🛠️ Tech Stack
- **Frontend**: Flutter + Riverpod  
- **Backend**: Supabase (Auth, Postgres, pgvector, Edge Functions)  
- **AI**: Google Gemini (with RAG)  
- **Scheduler**: Supabase cron jobs for reminders
